Assystonildo

Política de Privacidade

Documento de privacidade da extensão Assystonildo, destinada ao apoio operacional no uso do assystweb do TRT3.

Última atualização: 27 de abril de 2026

1. Escopo

A extensão Assystonildo foi criada para auxiliar o uso do sistema assystweb do TRT3. Ela atua apenas nos domínios explicitamente declarados em seu manifesto e utiliza somente as informações necessárias para executar seus próprios recursos.

2. Dados Tratados

A extensão pode processar dados exibidos nas páginas acessadas pelo usuário dentro dos sistemas suportados, como:

  • identificadores de chamados
  • nomes exibidos na interface
  • informações de contato
  • dados funcionais ou organizacionais visíveis no contexto do atendimento
  • identificadores funcionais usados para consultar relatórios MA, como login ou shortcode
  • conteúdo de comunicações preparadas pelo usuário dentro do assystweb, como destinatário, assunto, corpo da mensagem e anexos selecionados do chamado

Esses dados são utilizados exclusivamente para fornecer os recursos da própria extensão.

3. Finalidade de Uso

Os dados tratados são usados apenas para:

  • exibir histórico contextual de chamados
  • apresentar detalhes complementares do atendimento
  • facilitar acesso a contatos e informações relacionadas
  • registrar comunicações com usuários dentro do próprio fluxo do assystweb
  • simplificar tarefas operacionais dentro do fluxo de uso do assystweb

4. Compartilhamento de Dados

Os dados não são vendidos.

Os dados não são compartilhados com terceiros para publicidade, marketing, perfilização ou revenda.

A extensão pode se comunicar com os seguintes serviços somente para executar suas funcionalidades:

  • https://assystweb.trt3.jus.br/, para consultar dados e anexos do assystweb com a sessão autenticada do usuário.
  • https://integradorpessoal.trt3.jus.br/, para consultar contatos e informações organizacionais relacionadas ao atendimento.
  • https://ma-registrologins-396866479628.southamerica-east1.run.app/, para baixar pacotes de dados MA usados localmente pela extensão.
  • https://script.google.com/, quando a fonte MA estiver publicada via Google Apps Script.
  • https://docs.google.com/ e https://*.googleusercontent.com/, quando a fonte MA estiver configurada em Google Sheets.

Quando recursos baseados em Google Sheets ou Apps Script são usados, o navegador pode enviar cookies de sessão do Google para autenticar o acesso do próprio usuário a esses recursos. A extensão não usa esses dados para publicidade, venda, perfilização ou qualquer finalidade fora do propósito único descrito acima. O uso de informações recebidas de APIs ou serviços Google segue a Política de Dados do Usuário da Chrome Web Store, incluindo os requisitos de Uso Limitado.

5. Armazenamento

Quando necessário, a extensão pode armazenar configurações locais no navegador do usuário por meio da API storage do Chrome.

A extensão também pode armazenar cache técnico local em IndexedDB para evitar baixar novamente pacotes MA já sincronizados. Esse cache permanece no navegador do usuário.

6. Publicidade

A extensão não exibe anúncios.

7. Segurança

A extensão solicita apenas as permissões estritamente necessárias ao seu funcionamento e opera somente nos domínios definidos no manifesto.

A extensão não solicita acesso a todas as páginas visitadas, não usa activeTab e não acessa localhost.

8. Contato

Para dúvidas sobre esta política de privacidade ou sobre a extensão, utilize o canal de contato informado pelo mantenedor da extensão.
